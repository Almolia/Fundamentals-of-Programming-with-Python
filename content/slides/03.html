مقادیر، متغیرها، عملگرها، تبدیل نوع و ورودی/خروجی

متغیرها و مقادیر

متغیر ها در پایتون یک سری نام های نمادینی هستند که به مقادیر یا اشیا در حافظه اشاره میکنند که میتوانند انواع مختلفی داشته باشند.

برای درک بهتر این موضوع، آشپزخانه را تصور کنید. شما برای پختن یک کیک به مواد اولیه مختلفی مانند آرد، شکر و شیر نیاز دارید. آیا این مواد را روی سطح میز پخش می‌کنید؟ مسلماً نه. شما به طور غریزی از ظرف‌های مختلفی استفاده می‌کنید: کاسه‌ای برای آرد، ظرفی برای شکر و لیوانی برای شیر. این ظرف‌ها به شما کمک می‌کنند تا مواد را به صورت منظم نگهداری کرده، مقدار آن‌ها را اندازه بگیرید و در زمان مناسب از آن‌ها استفاده کنید. در برنامه‌نویسی نیز دقیقاً همین نیاز وجود دارد. ما با مقادیر مختلفی (اعداد، متن‌ها، لیست‌ها) سر و کار داریم و برای اینکه بتوانیم این مقادیر را در مراحل مختلف برنامه مدیریت کنیم، به «ظرف» یا نگهدارنده‌ای نیاز داریم. متغیرها دقیقاً نقش همین ظرف‌ها را بازی می‌کنند؛ هر متغیر نامی منحصر به فرد دارد و مقداری را در خود نگه می‌دارد، درست مانند ظرف‌های آشپزخانه که هرکدام برای نگهداری نوع خاصی از مواد مناسب‌تر هستند.

حالا به قفسه ادویه‌جات فکر کنید. برای اینکه به راحتی فلفل را از زردچوبه تشخیص دهید، روی هر ظرف یک برچسب می‌زنید و نام آن را می‌نویسید. در برنامه‌نویسی هم ما برای متغیرها نام‌گذاری می‌کنیم تا بتوانیم به راحتی به مقداری که درونشان است دسترسی پیدا کنیم. هر بار که به فلفل نیاز دارید، به سراغ ظرفی با برچسب «فلفل» می‌روید و مقداری از آن را برمی‌دارید. ممکن است مقدار فلفل داخل ظرف کم یا زیاد شود، اما ظرف و برچسب آن همیشه سر جای خود باقی است. متغیرها نیز به همین شکل عمل می‌کنند: نام آن‌ها ثابت است، اما مقداری که در خود نگه می‌دارند می‌تواند در طول اجرای برنامه تغییر کند.

# تعریف متغیرها
x = 10
name = "Ali"
is_student = True
print(x, name, is_student)

خروجی:

10 Ali True

نکات کلیدی:

  • نام متغیرها فقط می‌توانند ترکیبی از اعداد، حروف و علامت (_) باشند.
  • متغیر در پایتون حتما باید با یک حرف یا علامت (_) شروع شود.
  • مثال مجاز: my_variable, var123, _myVar
  • مثال غیرمجاز: 123var, my-var, my var
  • نوع داده می‌تواند عددی، متنی یا منطقی باشد.
  • نیازی به تعریف نوع داده نیست، پایتون به طور خودکار آن را تشخیص می‌دهد.
  • زبان پایتون به بزرگ و کوچک بودن حروف حساس است (Case-Sensitive)، یعنی دو متغیر A و a متفاوت هستند.

کلمات رزرو شده در پایتون (Reserved Words)

کلمات رزرو شده در پایتون، کلماتی هستند که معنای خاصی دارند و نمی‌توان از آن‌ها به عنوان نام متغیر یا تابع استفاده کرد.

💡 نکته:

این کلمات بخشی از ساختار زبان هستند. مثلاً if برای شرط، for برای حلقه، و def برای تعریف تابع استفاده می‌شود. 

import keyword
print(keyword.kwlist)

خروجی:

['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await',
 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except',
 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda',
 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while',
 'with', 'yield']

توضیح: این کلمات برای وظایف خاصی در پایتون استفاده می‌شوند. به عنوان مثال:

  • if, else, elif → برای تصمیم‌گیری (شرط)
  • for, while → برای اجرای تکرار (حلقه)
  • def → برای تعریف توابع
  • True, False, None → برای مقادیر خاص

نکات کلیدی:

  • نمی‌توان از کلمات رزرو شده به عنوان نام متغیر یا تابع استفاده کرد.
  • برای مشاهده لیست کامل می‌توانید از ماژول keyword استفاده کنید.

انواع داده‌ها در پایتون (Data Types)

در پایتون، هر مقدار دارای یک نوع داده است که تعیین می‌کند چه نوع عملیاتی می‌توان روی آن انجام داد. پایتون به طور خودکار نوع داده را تشخیص می‌دهد.

💡 مثال روزمره:

دوباره به ظروف آشپزخانه فکر کنید. هر ظرفی شکل و کاربرد خاص خود را دارد. شما برای آب از «بطری»، برای تخم‌مرغ از «شانه تخم‌مرغ»، برای سوپ از «کاسه» و برای برنج از «بشقاب» استفاده می‌کنید. هیچ‌گاه تلاش نمی‌کنید سوپ را در شانه تخم‌مرغ حمل کنید یا یک سیب کامل را در نمکدان قرار دهید! نوع ظرف، مشخص می‌کند که چه چیزی می‌تواند درون آن قرار بگیرد و چه کاری می‌توان با آن انجام داد.

انواع داده‌ها در پایتون دقیقاً همین نقش را دارند. نوع int ظرفی برای اعداد صحیح، و نوع str ظرفی برای متن است. این «نوع» مشخص می‌کند که چه مقداری را می‌توان در متغیر ذخیره کرد و چه عملیاتی (مانند جمع ریاضی یا چسباندن متن‌ها به هم) روی آن مجاز است. 

# انواع اصلی داده‌ها در پایتون
x1 = 10              # int → عدد صحیح
x2 = 3.14            # float → عدد اعشاری
x3 = "Hello"         # str → رشته
x4 = True            # bool → مقدار منطقی
x5 = [1, 2, 3]       # list → لیست
x6 = (1, 2, 3)       # tuple → تاپل
x7 = {1, 2, 3}       # set → مجموعه
x8 = {"name": "Ali", "age": 21}   # dict → دیکشنری
x9 = None            # NoneType → مقدار تهی

خروجی بررسی نوع داده‌ها:

print(type(10))            # <class 'int'>
print(type(3.14))          # <class 'float'>
print(type("Hello"))       # <class 'str'>
print(type(True))          # <class 'bool'>
print(type([1, 2, 3]))     # <class 'list'>
print(type((1, 2, 3)))     # <class 'tuple'>
print(type({1, 2, 3}))     # <class 'set'>
print(type({"x": 1}))      # <class 'dict'>
print(type(None))          # <class 'NoneType'>

نکات کلیدی:

  • اعداد صحیح (int): این نوع داده برای ذخیره اعداد کامل و بدون اعشار (مانند سن، تعداد دانشجویان، یا شماره یک صفحه) استفاده می‌شود.
  • اعداد اعشاری (float): برای ذخیره اعدادی که بخش کسری دارند (مانند میانگین نمرات، قیمت یک کالا، یا اندازه‌گیری‌های دقیق علمی) به کار می‌رود.
  • رشته‌ها (str): برای نگهداری داده‌های متنی مانند نام، آدرس، یا هر پیام دیگری استفاده می‌شود. رشته‌ها همیشه داخل علامت نقل قول (" " یا ' ') قرار می‌گیرند.
  • مقادیر منطقی (bool): این نوع داده فقط دو مقدار می‌پذیرد: True (درست) یا False (نادرست). از آن برای تصمیم‌گیری و کنترل جریان برنامه در شرط‌ها استفاده می‌شود.
  • انواع داده مرکب: انواع داده‌ای مانند لیست (list)، تاپل (tuple) و دیکشنری (dict) که برای نگهداری مجموعه‌ای از مقادیر به کار می‌روند، در جلسات آینده به تفصیل بررسی خواهند شد.

عملگرهای ریاضی در پایتون ➕➖✖️➗

عملگرهای ریاضی برای انجام محاسبات عددی در پایتون استفاده می‌شوند. این عملگرها مانند دکمه‌های ماشین‌حساب هستند و به ما کمک می‌کنند تا عملیات مختلف روی اعداد انجام دهیم.

جدول عملگرهای ریاضی در پایتون:

عملگر توضیح مثال نتیجه
+ جمع 5 + 3 8
- تفریق 10 - 4 6
* ضرب 7 * 2 14
/ تقسیم (اعشاری) 9 / 2 4.5
// تقسیم صحیح 9 // 2 4
% باقیمانده 9 % 2 1
** توان 2 ** 3 8 
# مثال از عملگرهای ریاضی
x = 9
y = 2

print(x + y)   # جمع: 11
print(x - y)   # تفریق: 7
print(x * y)   # ضرب: 18
print(x / y)   # تقسیم: 4.5
print(x // y)  # تقسیم صحیح: 4
print(x % y)   # باقیمانده: 1

🔢 ترتیب انجام عملیات (Order of Operations):

پایتون مانند ریاضیات، قوانین اولویت‌بندی دارد:

  1. ( ) → پرانتزها (بالاترین اولویت)
  2. ** → توان
  3. * , / , // , % → ضرب و تقسیم (از چپ به راست)
  4. + , - → جمع و تفریق (از چپ به راست)

🧠 تمرین:

نتیجه‌ی عبارت زیر چیست؟

(4 + 2) * 2 ** 3

الف) 48    ب) 36    ج) 24    د) 64

نکات کلیدی:

  • عملگرهای ریاضی برای انجام محاسبات عددی استفاده می‌شوند.
  • برای گرفتن بخش صحیح از تقسیم از // و برای باقیمانده از % استفاده کنید.
  • پرانتزها همیشه بالاترین اولویت را در محاسبات دارند.

عملگرهای مقایسه‌ای در پایتون ⚖️

عملگرهای مقایسه‌ای (Comparison Operators) برای مقایسه دو مقدار استفاده می‌شوند. نتیجه‌ی این مقایسه همیشه یک مقدار بولی است: True یا False.

💡 مثال روزمره:

وقتی می‌پرسید «آیا امروز دمای هوا بیشتر از دیروز است؟»، در واقع یک مقایسه انجام می‌دهید! پایتون هم همین کار را با اعداد، رشته‌ها و دیگر داده‌ها انجام می‌دهد.

جدول عملگرهای مقایسه‌ای:

عملگر توضیح مثال نتیجه
== بررسی برابری 5 == 5 True
!= بررسی نابرابری 5 != 3 True
> بزرگ‌تر 7 > 4 True
< کوچک‌تر 2 < 9 True
>= بزرگ‌تر یا مساوی 5 >= 5 True
<= کوچک‌تر یا مساوی 3 <= 4 True 
# مثال از عملگرهای مقایسه‌ای
x = 10
y = 5

print(x == y)   # False
print(x != y)   # True
print(x > y)    # True
print(x <= 5)   # False

🧠 تمرین:

نتیجه‌ی کد زیر چیست؟

a = 7
b = 10
print(a * 2 > b)

الف) True    ب) False    ج) Error

نکات کلیدی:

  • عملگرهای مقایسه‌ای مقدار بولی True یا False برمی‌گردانند.
  • از == برای برابری و از != برای نابرابری استفاده کنید.
  • این عملگرها در ساختارهای شرطی (if) و حلقه‌ها بسیار پرکاربردند.

عملگرهای منطقی در پایتون 🧠

عملگرهای منطقی (and, or, not) برای ترکیب چند عبارت شرطی استفاده می‌شوند. خروجی آن‌ها نیز همیشه یک مقدار بولی (True یا False) است.

💡 مثال روزمره:

برای رفتن به سینما، باید «بلیط داشته باشید» و «فیلم شروع نشده باشد». هر دو شرط باید برقرار باشند (عملگر and). برای ناهار، «پیتزا می‌خورید» یا «ساندویچ» (عملگر or). و اگر هوا بارانی نباشد به پیاده‌روی می‌روید (عملگر not).

جدول عملگرهای منطقی:

عملگر توضیح مثال نتیجه
and اگر هر دو شرط True باشند، نتیجه True است (5 > 3) and (2 < 4) True
or اگر حداقل یکی از شرط‌ها True باشد، نتیجه True است (5 < 3) or (2 < 4) True
not نتیجه را معکوس می‌کند (True به False و برعکس) not (5 < 3) True 
# مثال از عملگرهای منطقی
age = 22
has_license = True

# آیا سن فرد بزرگتر از 18 است و گواهینامه دارد؟
print(age > 18 and has_license)

# آیا فرد زیر 18 سال است یا گواهینامه ندارد؟
print(age < 18 or not has_license)

نمونه خروجی:

True
False

🧠 تمرین:

نتیجه‌ی کد زیر چیست؟

print((10 == 10) or not (5 > 1))

الف) True    ب) False    ج) Error

نکات کلیدی:

  • عملگر and تنها زمانی True برمی‌گرداند که تمام شرط‌ها درست باشند.
  • عملگر or زمانی True برمی‌گرداند که حداقل یکی از شرط‌ها درست باشد.
  • عملگر not یک عبارت بولی را نقیض (معکوس) می‌کند.
  • این عملگرها برای ساختن شرط‌های پیچیده در دستور if حیاتی هستند.

عملگرهای عضویت در پایتون 🔍

عملگرهای عضویت (in و not in) برای بررسی اینکه آیا یک مقدار خاص در یک دنباله (مانند رشته، لیست یا تاپل) وجود دارد یا خیر، استفاده می‌شوند.

💡 مثال روزمره:

این عملگرها مانند جستجو کردن هستند. مثلاً بررسی می‌کنید آیا حرف «ب» در کلمه «کتاب» وجود دارد (in)؟ یا آیا «شکر» در لیست خرید شما وجود ندارد (not in)؟

جدول عملگرهای عضویت:

عملگر توضیح مثال نتیجه
in اگر مقدار در دنباله موجود باشد، نتیجه True است 'a' in 'banana' True
not in اگر مقدار در دنباله موجود نباشد، نتیجه True است 'z' not in 'banana' True 
# مثال از عملگرهای عضویت
message = "Hello Python World"
numbers = [10, 20, 30, 40]

# آیا کلمه "Python" در پیام وجود دارد؟
print("Python" in message)

# آیا عدد 50 در لیست اعداد وجود ندارد؟
print(50 not in numbers)

نمونه خروجی:

True
True

🧠 تمرین:

نتیجه‌ی کد زیر چیست؟

my_list = ["apple", "banana"]
print("Apple" in my_list)

الف) True    ب) False    ج) Error

(نکته: پایتون به بزرگی و کوچکی حروف حساس است)

نکات کلیدی:

  • عملگر in برای بررسی وجود یک عضو در یک مجموعه استفاده می‌شود.
  • عملگر not in برای اطمینان از عدم وجود یک عضو در مجموعه به کار می‌رود.
  • این عملگرها روی انواع داده دنباله‌ای مانند رشته‌ها، لیست‌ها و تاپل‌ها کار می‌کنند.
  • مقایسه در این عملگرها به بزرگی و کوچکی حروف حساس (Case-Sensitive) است.

تبدیل نوع داده‌ها (Type Casting) 🔄

در پایتون می‌توان نوع داده‌ها را به یکدیگر تبدیل کرد. این کار به شما کمک می‌کند تا مقادیر مختلف مانند عدد و رشته را در محاسبات یا نمایش با هم ترکیب کنید.

💡 مثال روزمره:

فرض کنید می‌خواهید عدد 20 را کنار رشته "سال" چاپ کنید. چون نوع داده‌ها متفاوت است، باید عدد را به رشته تبدیل کنید. مثل ترجمه کردن زبان‌ها برای فهم مشترک!

# تبدیل رشته به عدد برای محاسبه
age_str = "25"
age_num = int(age_str)
print("سن شما 5 سال دیگر:", age_num + 5)

# تبدیل عدد به رشته برای نمایش
score = 100
print("امتیاز شما: " + str(score))

نمونه خروجی:

سن شما 5 سال دیگر: 30
امتیاز شما: 100

⚠️ توجه: اگر سعی کنید رشته‌ای غیرعددی را با int() به عدد تبدیل کنید، پایتون خطا می‌دهد:

int("hello")  # ❌  خطای ValueError می‌دهد

نکات کلیدی:

  • توابع اصلی: int(), float(), str(), bool().
  • تبدیل نوع برای ترکیب داده‌ها در محاسبات یا نمایش ضروری است.
  • برای جلوگیری از خطا، همیشه از قابل تبدیل بودن داده مطمئن شوید.

ورودی و خروجی (Input / Output) 🖥️

تعامل با کاربر از طریق تابع input() برای دریافت ورودی و تابع print() برای نمایش خروجی انجام می‌شود.

📘 نکته بسیار مهم:
خروجی تابع input() همیشه یک رشته (str) است.
برای انجام عملیات ریاضی، باید آن را با int() یا float() به نوع عددی تبدیل کنید.

# دریافت نام و نمایش آن
name = input("نام شما چیست؟ ")
print("سلام،", name, "!")

# دریافت سن و محاسبه
age_str = input("چند سال دارید؟ ")
age = int(age_str)
print("سال آینده شما", age + 1, "ساله خواهید شد.")

🧩 خروجی نمونه:

نام شما چیست؟ سارا
سلام، سارا !
چند سال دارید؟ 21
سال آینده شما 22 ساله خواهید شد.

🧠 تمرین:

برنامه‌ای بنویسید که نام کاربر و سال تولد او را بگیرد و سن تقریبی او را محاسبه و نمایش دهد.

راهنما: سن = سال فعلی - سال تولد. (سال فعلی را 1404 در نظر بگیرید)

نکات کلیدی:

  • input() همیشه مقدار را به صورت رشته (str) برمی‌گرداند.
  • برای انجام عملیات عددی باید خروجی input() را به int() یا float() تبدیل کرد.
  • print() برای نمایش خروجی به کار می‌رود و می‌تواند چند مقدار را با کاما چاپ کند.

🎉 تبریک! شما الفبای برنامه‌نویسی را آموختید!

شما با مفاهیم حیاتی مانند متغیرها برای ذخیره اطلاعات، انواع داده‌ها برای درک مقادیر مختلف، و مجموعه‌ای از عملگرها برای انجام محاسبات، مقایسه و تصمیم‌گیری آشنا شدید. همچنین یاد گرفتید چگونه با کاربر تعامل کنید و برنامه‌های ساده بسازید.

مروری بر دستاوردهای شما:

  • ذخیره اطلاعات: شما اکنون می‌دانید چگونه با استفاده از متغیرها داده‌ها را نام‌گذاری و ذخیره کنید.
  • درک داده‌ها: با انواع داده اصلی (عدد، متن، منطقی) و کاربرد هرکدام آشنا شدید.
  • انجام عملیات: یاد گرفتید چگونه با عملگرهای ریاضی، مقایسه‌ای و منطقی محاسبات و شرط‌های پیچیده بسازید.
  • تعامل با کاربر: می‌توانید با استفاده از input() داده دریافت کرده و با print() نتایج را نمایش دهید.

🚀 قدم بعدی چیست؟ حالا نوبت ترکیب این مفاهیم است! سعی کنید یک برنامه کوچک بنویسید. مثلاً یک ماشین حساب ساده که دو عدد از کاربر بگیرد (ورودی و تبدیل نوع)، آن‌ها را با هم جمع کند (عملگر ریاضی) و نتیجه را همراه با یک پیام مناسب نمایش دهد (خروجی). شما تمام ابزارهای لازم برای حل مسائل ساده را در اختیار دارید. به دنیای حل مسئله با کد خوش آمدید!
