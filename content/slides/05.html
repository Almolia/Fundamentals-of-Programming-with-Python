ساختار تکرار: حلقه‌ها

یادگیری نحوه تکرار دستورات با حلقه‌های for و while

۱. حلقه‌ها (Loops) در پایتون

در پایتون از حلقه‌ها برای تکرار بهینه اقدامات استفاده می‌شود. اگر حلقه‌ها وجود نداشتند، مجبور بودیم یک عمل تکراری را بارها و بارها با کدهای جداگانه بنویسیم که هم خسته‌کننده و هم مستعد خطا بود. حلقه‌ها این امکان را می‌دهند که وظایف تکراری به صورت خودکار انجام شوند و تنها با چند خط کد، عملیات تکراری را مدیریت کنیم.
در بسیاری از کاربردهای عملی مانند تحلیل داده‌ها، پردازش فایل‌ها یا انجام شبیه‌سازی‌ها، نیاز داریم یک کار را بارها انجام دهیم. استفاده از حلقه‌ها باعث صرفه‌جویی در زمان، افزایش خوانایی برنامه و نگهداری آسان‌تر کد می‌شود. در واقع، حلقه‌ها یکی از پایه‌های اصلی برنامه‌نویسی هستند چون اجرای تکرارشونده را ممکن می‌کنند و برنامه‌ها را قدرتمند و منعطف می‌سازند.
انواع اصلی حلقه‌ها عبارتند از حلقه‌های for (شمارش روی آیتم‌ها) و حلقه‌های while (بر اساس شرایط).

🔍 تشبیه: لیست وظایف

حلقه for: "به ازای هر کار در لیست وظایف، آن را انجام بده و تیک بزن." (شما می‌دانید دقیقاً چند کار دارید).
حلقه while: "تا زمانی که انرژی داری، به کار کردن ادامه بده." (شما نمی‌دانید دقیقاً چه زمانی انرژی‌تان تمام می‌شود).

📌 نکات کلیدی

  • حلقه‌ها کد را کوتاه و خوانا نگه می‌دارند (اصل DRY).
  • for برای تکرار روی دنباله‌ها (تعداد مشخص).
  • while برای تکرار بر اساس شرط (تعداد نامشخص).

❓ خودآزمایی

کدام نوع حلقه (for یا while) برای برنامه‌ای که از کاربر رمز عبور می‌پرسد تا زمانی که آن را درست وارد کند، مناسب‌تر است؟

۲. حلقه for

حلقه for یکی از پرکاربردترین ابزارها برای تکرار (iterate) روی یک sequence (مانند list، tuple، string یا range) است. این حلقه به شما اجازه می‌دهد اعضای یک لیست یا دنباله را یکی‌یکی بردارید و روی هر کدام عملی انجام دهید؛ مثلاً هر عضو را بررسی، چاپ یا علامت‌گذاری کنید.
زمانی که تعداد دفعات تکرار از قبل مشخص باشد، حلقه for بهترین انتخاب است. مثلاً اگر بخواهیم روی تمام اعضای یک لیست یا تعداد مشخصی تکرار انجام دهیم، این حلقه بسیار ساده و خواناست.
اگر بخواهید حلقه را دقیقاً به تعداد دلخواه اجرا کنید (مثلاً ۵ بار)، می‌توانید از تابع range() استفاده کنید. این تابع یک دنباله از اعداد تولید می‌کند و حلقه for روی آن حرکت می‌کند. برای نمونه، کد زیر ۵ بار اجرا می‌شود و در هر بار شماره مرحله را چاپ می‌کند: 

# حلقه for روی یک list
my_list = [1, 2, 3, 4]
for item in my_list:
    print(item)

# حلقه for روی یک string
my_string = "Python"
for char in my_string:
    print(char)

❓ خودآزمایی

یک لیست از ۵ اسم ایجاد کنید. با استفاده از حلقه for، به هر اسم عبارت "Bye " را اضافه کرده و آن را چاپ کنید.

۳. سینتکس حلقه for

🔹 سینتکس حلقه for

ساختار کلی حلقه for در پایتون به صورت for variable in sequence: است.
variable متغیری است که در هر بار تکرار مقدار جدیدی از sequence می‌گیرد.
sequence می‌تواند لیست، رشته، تاپل یا خروجی تابعی مثل range() باشد.
بدنه (body) حلقه شامل دستوراتی است که در هر بار تکرار اجرا می‌شوند.
می‌توان از متغیر شمارنده (مثلاً i در مثال‌ها) داخل بدنه برای انجام کارهایی مثل چاپ مقدار، محاسبه یا شرط استفاده کرد. 

# مثال ساده: چاپ مقدار و مربع آن با استفاده از شمارنده حلقه
for i in range(3):
    print(f"i = {i}, مربع = {i*i}")

for i in range(5):
    print(f"مرحله {i}")

۴. تکرار با Index در for

ما همچنین می‌توانیم از index عناصر در sequence برای تکرار استفاده کنیم. ایده کلیدی این است که ابتدا طول لیست را محاسبه کرده و سپس بر روی sequence در محدوده این طول تکرار کنیم.

my_list = ['a', 'b', 'c']
list_len = len(my_list)

for i in range(list_len):
    print(f"Index {i} has value: {my_list[i]}")

✅ خروجی:

Index 0 has value: a
Index 1 has value: b
Index 2 has value: c

💡 نکته: استفاده از for item in my_list (روش اسلاید قبلی) معمولاً «پایتونیک‌تر» (Pythonic) و خواناتر از تکرار بر اساس index است.

❓ خودآزمایی

با استفاده از range و len، کدی بنویسید که فقط آیتم‌های با اندیس زوج (0, 2, 4, ...) را از یک لیست چاپ کند.

۵. استفاده از تابع enumerate در حلقه for

گاهی لازم است در حین تکرار روی آیتم‌های یک لیست، هم‌زمان به مقدار و شماره‌ی اندیس آن‌ها دسترسی داشته باشیم. در این مواقع به جای استفاده از range و len، می‌توانیم از تابع enumerate() استفاده کنیم.
این تابع به طور خودکار در هر تکرار، دو مقدار برمی‌گرداند: index (شماره عنصر) و value (مقدار عنصر). این کار باعث می‌شود کد ساده‌تر، خواناتر و «پایتونیک‌تر» باشد. 

# استفاده از enumerate برای گرفتن همزمان index و مقدار
my_list = ['apple', 'banana', 'cherry']

for index, value in enumerate(my_list):
    print(f"Index {index} → Value: {value}")

✅ خروجی:

Index 0 → Value: apple
Index 1 → Value: banana
Index 2 → Value: cherry

💡 نکته: تابع enumerate() به صورت پیش‌فرض از ۰ شروع می‌کند، اما می‌توان با پارامتر دوم، نقطه‌ی شروع را تغییر داد (مثلاً enumerate(my_list, start=1)).

❓ خودآزمایی

یک لیست از نام شهرها بنویسید و با استفاده از enumerate()، شماره‌ی هر شهر را از ۱ شروع کرده و در قالب "شهر ۱: تهران" چاپ کنید.

۵. حلقه while

در Python، حلقه while برای اجرای مکرر یک بلوک از دستورات تا زمانی که یک شرط معین برقرار باشد، استفاده می‌شود. وقتی شرط false شود، خط بلافاصله بعد از حلقه در برنامه اجرا می‌شود.

# حلقه تا زمانی اجرا می‌شود که cnt < 3 باشد
cnt = 0
while (cnt < 3):
    cnt = cnt + 1
    print("Hello Geek")

print("End of loop")

✅ خروجی:

Hello Geek
Hello Geek
Hello Geek
End of loop

❓ خودآزمایی

یک حلقه while بنویسید که اعداد را از ۱۰ به صورت معکوس بشمارد و چاپ کند (10, 9, 8, ... 1). فراموش نکنید که شمارنده را کم کنید!

۶. حلقه infinite while

اگر بخواهیم یک بلوک کد به تعداد نامحدود (infinite) اجرا شود، می‌توانیم از حلقه while استفاده کنیم. کد زیر از while True استفاده می‌کند، به این معنی که حلقه تا ابد اجرا می‌شود مگر اینکه با دستور break از آن خارج شویم.

# این حلقه تا زمانی که به break نرسد، متوقف نمی‌شود
count = 0
while True:
    count += 1
    print(f"Iteration {count}")
    if count >= 3:
        break # خروج از حلقه بی‌نهایت

✅ خروجی:

Iteration 1
Iteration 2
Iteration 3

💡 هشدار: هنگام استفاده از while True همیشه مطمئن شوید که یک شرط break مشخص دارید، در غیر این صورت برنامه شما هنگ می‌کند.

❓ خودآزمایی

یک حلقه while True بنویسید که از کاربر یک عدد بگیرد. اگر کاربر عدد ۰ را وارد کرد، با دستور break از حلقه خارج شوید.

۷. حلقه‌های تودرتو (Nested Loops)

زبان Python اجازه می‌دهد از یک حلقه در داخل حلقه‌ای دیگر استفاده شود که به آن حلقه تودرتو (nested loop) می‌گویند. می‌توان هر نوع حلقه‌ای را درون هر نوع حلقه دیگری قرار داد (مثلاً for داخل while یا برعکس).

# در این کد، حلقه درونی (j) به تعداد i (شماره تکرار حلقه بیرونی) اجرا می‌شود
for i in range(1, 4):      # حلقه بیرونی: i از 1 تا 3
    for j in range(i):     # حلقه درونی: j از 0 تا i-1
        print(i, end=" ")  # مقدار i را چاپ کن
    print()                # پس از اتمام حلقه درونی، به خط بعد برو

✅ خروجی:

1 
2 2 
3 3 3

📌 نکته

  • به ازای هر یک تکرار حلقه بیرونی، حلقه درونی تمام تکرارهای خود را کامل اجرا می‌کند.

❓ خودآزمایی

با استفاده از حلقه‌های تودرتو، جدول ضرب ۳x۳ (از ۱ تا ۳) را چاپ کنید.

۸. دستورات کنترل حلقه (Control Statements)

دستورات کنترل حلقه، اجرا را از توالی عادی خود تغییر می‌دهند. وقتی اجرا از یک scope خارج می‌شود، تمام اشیاء خودکاری که در آن scope ایجاد شده بودند، از بین می‌روند. پایتون از سه دستور کنترلی اصلی پشتیبانی می‌کند.

🔍 تشبیه: رانندگی در یک مسیر مسابقه

continue: شما یک پیچ را نادیده می‌گیرید و میان‌بر می‌زنید تا به دور بعدی برسید (پرش از این تکرار).
break: شما از پیت-استاپ (Pit Stop) استفاده می‌کنید و به طور کامل از مسابقه خارج می‌شوید (خروج کامل از حلقه).
pass: شما از خط شروع/پایان عبور می‌کنید اما کاری انجام نمی‌دهید، فقط به دور خود ادامه می‌دهید (دستور خالی).

📌 ۳ دستور کنترلی

  • Continue Statement: کنترل را به ابتدای حلقه بازمی‌گرداند.
  • Break Statement: کنترل را از حلقه خارج می‌کند.
  • Pass Statement: هیچ کاری انجام نمی‌دهد (یک دستور خالی).

۹. دستور continue

دستور continue در پایتون، کنترل را به ابتدای حلقه بازمی‌گرداند. این دستور برای رد شدن (skip) از تکرار فعلی حلقه و رفتن به تکرار بعدی استفاده می‌شود و برای زمانی مفید است که بخواهیم از شرایط خاصی عبور کنیم بدون اینکه حلقه خاتمه یابد.

# چاپ تمام حروف به جز 'e' و 's'
for letter in 'geeksforgeeks':
    if letter == 'e' or letter == 's':
        continue
    print(f"Current Letter : {letter}")

✅ خروجی:

Current Letter : g
Current Letter : k
Current Letter : f
Current Letter : o
Current Letter : r
Current Letter : g
Current Letter : k

❓ خودآزمایی

یک حلقه for بنویسید که روی اعداد ۱ تا ۱۰ تکرار کند. با استفاده از continue، کاری کنید که فقط اعداد فرد چاپ شوند (از اعداد زوج بپرید).

۱۰. دستور break

دستور break در پایتون کنترل را از حلقه خارج می‌کند. این دستور برای خروج زودهنگام (prematurely) از حلقه هنگام برآورده شدن یک شرط مشخص استفاده می‌شود و اجرای حلقه متوقف می‌شود.

# توقف حلقه به محض رسیدن به 'e' یا 's'
for letter in 'geeksforgeeks':
    if letter == 'e' or letter == 's':
        break
    print(f"Current Letter : {letter}")
print("End of loop")

✅ خروجی:

Current Letter : g
End of loop

❓ خودآزمایی

یک لیست از اعداد [1, 5, 9, -4, 10, 2] دارید. با حلقه for روی آن حرکت کنید و به محض رسیدن به اولین عدد منفی، حلقه را با break متوقف کنید.

۱۱. دستور pass

ما از دستور pass در پایتون برای نوشتن حلقه‌های خالی استفاده می‌کنیم. pass همچنین برای دستورات کنترلی، توابع و کلاس‌های خالی استفاده می‌شود. در این مثال، حلقه روی هر حرف در 'geeksforgeeks' تکرار می‌شود اما هیچ عملیاتی انجام نمی‌دهد.

s = 'geeksforgeeks'
for letter in s:
    # فعلا نمی‌خواهیم کاری انجام دهیم، اما یک بدنه لازم است
    pass

# پس از پایان حلقه، 'letter' آخرین مقدار خود را حفظ کرده است
print(f"Last Letter : {letter}")

✅ خروجی:

Last Letter : s

❓ کاربرد `pass`

از pass اغلب به عنوان یک (placeholder) استفاده می‌شود. زمانی که در حال طراحی ساختار کد خود هستید و می‌دانید به یک حلقه یا تابع نیاز دارید اما هنوز آماده نوشتن منطق آن نیستید، از pass استفاده می‌کنید تا از خطای SyntaxError جلوگیری کنید.

🎉 تبریک! شما مفاهیم حلقه‌ها را یاد گرفتید!

عالی بود! تا اینجا یاد گرفتید که چگونه از حلقه‌ها برای اجرای دستورات تکراری در پایتون استفاده کنید.
اکنون می‌دانید تفاوت بین حلقه‌های for و while چیست، چطور با range() تعداد تکرار را مشخص کنید، و با تابع enumerate() هم‌زمان به اندیس و مقدار عناصر دسترسی داشته باشید.
همچنین با حلقه‌های تودرتو و دستورات کنترلی مانند break، continue و pass آشنا شدید.

📘 مرور آموخته‌ها

  • حلقه for برای پیمایش دنباله‌ها و تکرار با تعداد مشخص استفاده می‌شود.
  • حلقه while برای تکرار تا زمانی که شرط برقرار است کاربرد دارد.
  • تابع range() برای تولید توالی عددی و کنترل دفعات تکرار مفید است.
  • enumerate() خوانایی کد را هنگام نیاز به index افزایش می‌دهد.
  • با دستورات break، continue و pass می‌توانید رفتار حلقه را کنترل کنید.

💡 در ادامه‌ی مسیر: وقتی با ساختارهای داده‌ای پیشرفته‌تر مثل list، dict و set آشنا شویم،
دوباره به حلقه‌ها برمی‌گردیم تا ببینیم چگونه می‌توان به شکل مؤثر روی این ساختارها پیمایش کرد.
